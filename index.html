EasySign

Introduction

This application is designed to help people learn sign language. The final version was tested and used by Zhenjiang Disabled Personsâ€™ Federation and received positive feedback.

Features

  • using learning curve to improve efficiency in memorizing various sign language gestures.
  • Translating text sentences into sign language videos by breaking a sentence down into words, fetching individual video clips demonstrating the given word, and synthesizing all clips into a single long video showing the given sentence in sign language.
